ğŸ§  **Project Overview**

Mental Health Check is a Flutter-based mobile application that enables students to evaluate their mental well-being through an interactive chatbot and structured assessments. The system uses Google Natural Language Processing (NLP) to analyze user responses and provide emotion-based insights and personalized wellness exercises.

The platform includes role-based administration for managing students, reviewing assessments, and generating mental health reports. A secure Flask backend and PostgreSQL database ensure data integrity, scalability, and privacy.
